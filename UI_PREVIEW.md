## Interactive Flow Example

### User Action: Add New Leave Policy
1. User scrolls to "Kebijakan Cuti" section
2. Clicks green "+ Tambah Cuti" button
3. New policy card appears with empty fields
4. User fills in:
   - Kode: CUTI_TAHUNAN
   - Nama: Cuti Tahunan
   - Deskripsi: (optional)
   - Jenis: Cuti Tahunan
   - Total Hari: 12
   - Pembatasan: Semua
   - Checkboxes: All checked
5. User scrolls down to "Simpan" button
6. Clicks "Simpan"
7. Success message appears

### User Action: Remove Leave Policy
1. User finds the policy to remove
2. Clicks red trash icon on the right
3. Policy card immediately disappears
4. User clicks "Update" to save changes
5. Changes are persisted
